
# Как получить Telegram API ID и API Hash

Для работы клиента Rumker (через `gram.js`) вам потребуется собственный API ID и Hash. Это позволяет приложению авторизовываться в Telegram от вашего имени.

## Пошаговая инструкция

1.  **Откройте сайт my.telegram.org**
    Перейдите по ссылке: [https://my.telegram.org](https://my.telegram.org)

2.  **Войдите в систему**
    Введите свой номер телефона (в международном формате) и нажмите Next. Вы получите сообщение в Telegram с кодом подтверждения. Введите этот код на сайте.

3.  **Перейдите в "API development tools"**
    После входа вы увидите несколько ссылок. Нажмите на **API development tools**.

4.  **Создайте новое приложение**
    Если у вас еще нет приложения, вы увидите форму **Create new application**. Заполните её:
    *   **App title**: Название вашего клиента (например, `Rumker Web`).
    *   **Short name**: Короткое имя (например, `rumker_web`).
    *   **URL**: Можно указать `http://localhost:5173` или ваш домен.
    *   **Platform**: Выберите `Web`.
    *   **Description**: Опционально (например, `My custom web client`).

    Нажмите **Create application**.

5.  **Скопируйте App configurations**
    Вы увидите экран с данными. Вам нужны два поля:
    *   **App api_id**: (число, например `123456`)
    *   **App api_hash**: (строка, например `0123456789abcdef0123456789abcdef`)

## Куда это вставить?

Откройте файл `.env` в корне проекта (если его нет, создайте, скопировав из `.env.example`) и добавьте:

```env
TELEGRAM_API_ID=123456
TELEGRAM_API_HASH=0123456789abcdef0123456789abcdef
```

**⚠️ Важно:** Никогда не передавайте эти ключи третьим лицам. Они дают доступ к созданию сессий от вашего имени.
