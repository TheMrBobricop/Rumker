## Критические баги

- [ ] **Upload 500** -- Supabase Storage возвращает 500 при загрузке файлов. Вероятно, не создан bucket `chat-media` или нет прав на запись. Проверить:
  1. Существует ли bucket `chat-media` в Supabase Dashboard -> Storage
  2. Настроены ли RLS-политики (разрешение на INSERT для authenticated пользователей)
  3. `SUPABASE_SERVICE_ROLE_KEY` корректен в `.env`
  4. Запустить `node setup-bucket.js` для автоматического создания bucket

- [ ] **Socket.io** -- серверная часть (`server/socket/index.ts`) -- заглушка. Сообщения не обновляются в реальном времени, нужен polling или полная реализация WebSocket.
